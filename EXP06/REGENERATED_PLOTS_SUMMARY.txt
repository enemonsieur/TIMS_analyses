==========================

Old (PLV-based):
  Raw:  Select channel with highest PLV vs GT per intensity (reselected each time)
  SASS: Decompose cleaned signal, select channel with highest PLV vs GT per intensity
  SSD:  Decompose raw signal, select component with highest PLV vs GT per intensity

RECOMMENDATIONS:
================

1. USE SNR FOR COMPONENT SELECTION (not PLV)
   ✓ SNR reveals true signal quality
   ✗ PLV is biased toward artifact phase-locking to GT

2. INTERPRET PLV WITH SNR CONTEXT
   - High PLV + high SNR = Real signal recovered
   - High PLV + low SNR = Artifact locking to GT (beware!)
   - Low PLV + high SNR = Real signal but less phase-locked

3. RECOGNIZE ARTIFACT SATURATION THRESHOLD
   - Below 30%: All methods can suppress broadband artifact
   - Above 30%: Only spectral methods (SSD, SASS partially) preserve signal
   - Above 40%: Raw method is unusable

4. FOR FUTURE ANALYSES
   - Always show both SNR and PLV
   - Don't rely on PLV alone for artifact assessment
   - Component selection: Use SNR, not PLV
   - Component ranking: Use multiple metrics together
